# Hvert medlem av Gruppe 2, skal følge dette#

- Hvert medlem skal ha sin egen branch, oppkalt etter kallenavn eller navnet ditt. Slik at man kan gjenkjenne hver sin branch.
- Når man er ferdig med å utføre en oppgave, skal man opprette en pull request, slik at all kode kan merges. 
- Man skal også samarbeide og snakke til vedkommende man ønsker å redigere koden til. For å unngå unødvendige feil.

Noen kjekke kommandoer, som er greit å vite om.

# For å opprette ny branch
git checkout -b <branch navn>

# For å skrifte branch.
git checkout <branch navn>

# For å pulle fra branch
git pull origin <branch navn>

# For å push(e) til branch
git push origin <branch navn>
